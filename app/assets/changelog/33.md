- Health, Damage, Healing, Xp, etc will now all scale much more smoothly with your characters level. This applies to both player characters, as well as creatures.
- Added more creature sound effects

- Reduced background processing while the world map is closed.
- Improved map and minimap quest marker performance.
- Kept quest marker priority behavior consistent.
- Improved cleanup when leaving or reloading the game world.
- Reduced processing overhead for nearby enemies and nameplates.
- Improved nameplate performance, especially when not visible.
- Reduced unnecessary tooltip updates when hovering over targets.
- Improved action bar cooldown performance.
